## How to use this guide

Cross-references are based on matching the four critical fitment dimensions:

1. **Case dimensions** (L × W × H must match the battery box)
2. **Terminal layout** (R+ vs L+, top post vs side post)
3. **Capacity (Ah)** within ±10%
4. **CCA rating** within ±10%

Always verify these on the actual battery before installation. Cross-reference tables are starting points, not guarantees of perfect interchangeability.

## Important notes

- **Warranty void if installed without verifying fitment** — case dimensions ±2mm, terminal layout exact match
- **L+ vs R+ matters** — the polarity layout designation (L for negative-on-left, R for positive-on-left) is critical, mismatch can cause cable strain or short circuits
- **CCA standards vary** — EN/SAE/JIS/DIN have different test methods. EN typically reads 10–15% lower than SAE for the same battery
- **Container size codes** — DIN H6/H7/H8/H9 vs Group 24/27/31/8D vs JIS D23/D26/E41 — verify against vehicle manual
